### Features
- User can import the full library or an individual component
- All the components are fully responsive

- Avatar
  - Circular Avatars
  - Square-Shaped Avatars
  - Avatars with status
- Alert
  - Standard Alerts
  - Alert with icons 
- Badge
  - Icons with badge
- Button
  - Solid buttons
  - Outline buttons
  - Link button
  - Image button
- Card
- Image
  - Responsive Image
  - Round Image
- Input
  - Standard Input
  - Validation Input
- List
  - Simple list
  - Stacked list
- Navigation
- Modal
